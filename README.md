# Hyprland Dotfiles

My personal Arch Linux + Hyprland configuration.

## Setup
- **OS**: Arch Linux
- **WM**: Hyprland
- **Terminal**: Kitty
- **Shell**: Zsh + Starship
- **Bar**: Waybar
- **File Manager**: Yazi
- **Wallpaper**: awww

## Install
Make sure you have `yay` installed, then run:
\```bash
git clone git@github.com:kyrdzik/hyprland_dotfiles.git ~/dotfiles
cd ~/dotfiles
chmod +x install.sh
./install.sh
\```

## Keybindings
| Key | Action |
|-----|--------|
| SUPER + E | Open Yazi file manager |
| SUPER + B | Open Waybar |
| XF86MonBrightnessUp/Down | Brightness +/- 10% |
| XF86AudioRaiseVolume/LowerVolume | Volume +/- 5% |
